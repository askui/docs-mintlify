## Basic Form Filling

```python
from askui import Agent, locators as loc


```

## Form Validation

Check for errors before and after submission using [`get()`](/04-reference/01-agent-frameworks/02-python/02-vision-agent-api/agent#get):

```python
# Fill form
agent.type("invalidemail", loc.Element().right_of(loc.Text("Email")))

# Check for inline validation
has_error = agent.get(  # See get() API reference
    "Is there an email validation error shown?",
    response_schema=bool
)

if has_error:
    error_msg = agent.get(
        "What is the email error message?",
        response_schema=str
    )
    print(f"Validation error: {error_msg}")
    
    # Fix the error
    agent.clear(loc.Element().right_of(loc.Text("Email")))
    agent.type("valid@example.com", loc.Element().right_of(loc.Text("Email")))
```

## Multi-Step Forms

Navigate through wizard-style forms:

```python
from askui import ResponseSchemaBase

class FormStep(ResponseSchemaBase):
    current_step: int
    total_steps: int
    step_name: str

# Step 1: Personal Information
agent.type("John", loc.Element().right_of(loc.Text("First Name")))
agent.type("Doe", loc.Element().right_of(loc.Text("Last Name")))
agent.click("Next")

# Check progress
progress = agent.get(
    "What is the current form step?",
    response_schema=FormStep
)
print(f"On step {progress.current_step} of {progress.total_steps}: {progress.step_name}")

# Step 2: Contact Details
agent.type("john@example.com", loc.Element().right_of(loc.Text("Email")))
agent.type("+1234567890", loc.Element().right_of(loc.Text("Phone")))
agent.click("Next")

# Step 3: Review and Submit
is_correct = agent.get(
    "Does the summary show 'John Doe' and 'john@example.com'?",
    response_schema=bool
)

if is_correct:
    agent.click("Submit")
else:
    agent.click("Back")
```

## Dynamic Forms

Handle forms that change based on selections:

```python
# Select dropdown option
agent.click(loc.Element().right_of(loc.Text("Country")))
agent.click("United States")

# Check if state field appeared
has_state_field = agent.get(
    "Is there a State field visible?",
    response_schema=bool
)

if has_state_field:
    agent.click(loc.Element().right_of(loc.Text("State")))
    agent.click("California")
```

## File Uploads

Handle file upload fields:

```python
# Click upload button
agent.click("Choose File")

# Type file path in system dialog
agent.type("/path/to/document.pdf")
agent.key("enter")

# Verify upload
file_uploaded = agent.get(
    "Is 'document.pdf' shown as uploaded?",
    response_schema=bool
)
```

## Form Data Extraction

Extract form data for verification:

```python
from typing import Dict, Any

class FormData(ResponseSchemaBase):
    name: str
    email: str
    phone: str
    newsletter: bool
    country: str

# Extract all form values
current_data = agent.get(
    "Extract all the form field values",
    response_schema=FormData
)

# Verify before submission
print(f"About to submit: {current_data.model_dump()}")
```

## Complex Form Patterns

### Address Autocomplete
```python
# Start typing address
agent.type(
    "123 Main",
    loc.Element().right_of(loc.Text("Address"))
)

# Wait for suggestions
agent.wait(1)

# Check if suggestions appeared
has_suggestions = agent.get(
    "Are address suggestions visible?",
    response_schema=bool
)

if has_suggestions:
    # Click first suggestion
    agent.click(loc.Text("123 Main Street, New York, NY"))
```

### Date Pickers
```python
# Click date field
agent.click(loc.Element().right_of(loc.Text("Date of Birth")))

# Navigate calendar
current_month = agent.get(
    "What month is shown in the calendar?",
    response_schema=str
)

# Navigate to correct month
while current_month != "March 2024":
    agent.click(loc.AiElement("calendar previous month button"))
    current_month = agent.get(
        "What month is shown in the calendar?",
        response_schema=str
    )

# Select date
agent.click("15")
```

### Radio Button Groups
```python
# Select from radio group
agent.click(
    loc.Element().right_of(loc.Text("Premium Plan"))
)

# Verify selection
selected_plan = agent.get(
    "Which plan is selected?",
    response_schema=str
)
```

## Form Testing Patterns

### Required Field Validation
```python
# Try submitting empty form
agent.click("Submit")

# Get all error messages
errors = agent.get(
    "What are all the error messages shown?",
    response_schema=List[str]
)

# Fill required fields based on errors
for error in errors:
    if "email" in error.lower():
        agent.type("test@example.com", loc.Element().right_of(loc.Text("Email")))
    elif "name" in error.lower():
        agent.type("Test User", loc.Element().right_of(loc.Text("Name")))
```

### Form Reset
```python
# Fill some fields
agent.type("test@example.com", loc.Element().right_of(loc.Text("Email")))

# Reset form
agent.click("Reset")

# Verify fields are empty
is_empty = agent.get(
    "Are all form fields empty?",
    response_schema=bool
)
```

## Best Practices

1. **Always validate before submission**
2. **Handle both client and server validation**
3. **Clear fields before re-entering data**
4. **Wait for dynamic content to load**
5. **Extract and verify data before final submission**
