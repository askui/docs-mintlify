## What is AskUI?

AskUI enables AI to control your desktop and mobile devices. We make Windows, macOS, Linux, Android and iOS accessible for AI agents by finding elements on screen and allowing [AgentOS](/06-agent-os/index) to execute agent commands on operating systems.

Three components work together:

- **[AgentOS](/06-agent-os/index)** — the runtime on your machine that captures screenshots, moves the mouse, and types on the keyboard.
- **A vision-language model** (e.g. Claude Sonnet) — the AI that looks at each screenshot and decides what to do next.
- **The [AskUI SDK](https://github.com/askui/python-sdk)** — the Python library that orchestrates the agent loop.
