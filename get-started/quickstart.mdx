By the end of this page, you'll watch AskUI open the Calculator app, click buttons, perform a calculation, and generate a report with screenshots — all autonomously.

## 1. Install Agent OS

If you haven't already, install Agent OS on your machine. Pick your platform:

<Tabs>
  <Tab title="Windows">
    ```powershell
    Invoke-WebRequest -Uri "https://files.askui.com/releases/Installer/25.11.1/AskUI-Suite-25.11.1-User-Installer-Win-AMD64-Web.exe" -OutFile "AskUI-Setup.exe"
    .\AskUI-Setup.exe
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    curl -o AskUI-Setup.run https://files.askui.com/releases/Installer/25.11.1/AskUI-Suite-25.11.1-User-Installer-MacOS-ARM64-Web.run
    chmod +x AskUI-Setup.run && ./AskUI-Setup.run
    ```
    <Warning>Apple Silicon only (M1/M2/M3/M4).</Warning>
  </Tab>
  <Tab title="Linux">
    ```bash
    curl -o AskUI-Setup.run https://files.askui.com/releases/Installer/25.11.1/AskUI-Suite-25.11.1-User-Installer-Linux-AMD64-Web.run
    chmod +x AskUI-Setup.run && ./AskUI-Setup.run
    ```
    <Warning>X11 only. No Wayland support.</Warning>
  </Tab>
</Tabs>

Need more options? See the full [Installation guide](/agent-os/installation).

## 2. Install the Python SDK

```bash
pip install askui[all]
```

## 3. Set Your Credentials

Sign up at [hub.askui.com](https://hub.askui.com) to get your **Workspace ID** and **Access Token**. These are needed for AskUI's cloud AI models.

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    export ASKUI_WORKSPACE_ID="your-workspace-id"
    export ASKUI_TOKEN="your-access-token"
    ```
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    $env:ASKUI_WORKSPACE_ID="your-workspace-id"
    $env:ASKUI_TOKEN="your-access-token"
    ```
  </Tab>
</Tabs>

<Tip>
Add these to your shell profile (`.bashrc`, `.zshrc`) or system environment variables so they persist across sessions.
</Tip>

## 4. Verify Agent OS Is Running

```bash
curl http://localhost:23000
```

If you get a response, you're good. If you get "connection refused", start Agent OS from your system tray or application launcher.

## 5. Run the Calculator Demo

Clone the demo project and run it:

```bash
git clone https://github.com/askui/AskUI-Demo-Project.git
cd AskUI-Demo-Project
pip install -r requirements.txt
python main.py tasks/calculator_test.csv
```

**Now watch your screen.** The agent will:

1. Open the Calculator app
2. Click the number buttons (2, 0, -, 1, 0)
3. Press equals
4. Verify the result is 10
5. Take a screenshot at each step

## 6. Check the Results

When the run finishes, look in the `agent_workspace/` folder:

```
agent_workspace/
└── 2026-03-18_14-50-00/
    └── calculator_test/
        ├── calculator_test_report.md   ← Detailed step-by-step report
        ├── step1_calculator_opened.png ← Screenshot evidence
        ├── step2_click_2.png
        └── ...
```

Open the markdown report to see every step, what happened, and whether it matched the expected result.

## What Just Happened?

Here's what ran under the hood:

1. **`main.py`** read the CSV task file — each row is a test step with an expected result
2. **The Python SDK** sent each step to AskUI's cloud AI model, which looked at the screen and decided what to click
3. **Agent OS** executed the clicks and keystrokes on your actual desktop
4. **Screenshots** were captured at each step and saved as evidence
5. **A report** was generated summarizing pass/fail for each step

```
[CSV Task File] → [Python SDK] → [AI Model] → [Agent OS] → [Your Screen]
                                                    ↓
                                              [Screenshots + Report]
```

## Try It Yourself

**Change the calculation:** Open `tasks/calculator_test.csv` in any text editor. Change the numbers and expected result. Run again.

**Write a plain-text task:** Create a new file `tasks/my_test.txt` with a natural language instruction:

```
Open the Notes app and type "Hello from AskUI"
```

Then run:

```bash
python main.py tasks/my_test.txt
```
