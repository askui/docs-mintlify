# AskUI-ShowProcess

### SYNOPSIS

Shows running AskUI processes.

### DESCRIPTION

The AskUI-ShowProcess cmdlet displays information about currently running AskUI processes
including process IDs, names and status.

### EXAMPLES

#### EXAMPLE 1

```powershell
AskUI-ShowProcess
Lists all running AskUI processes.
```
