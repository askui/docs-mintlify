---
title: "System Requirements"
description: "Hardware and software requirements for running AskUI"
---

AskUI is compatible with:

- **Windows**: Windows 10 version 22H2+ (64-bit), Windows 11 (64-bit), Windows Server 2022/2025 (64-bit)
- **macOS**: macOS 10.15 (Catalina) or newer
- **Linux**: Ubuntu 18.04 or newer

<Note>
**Python 3.10 or newer** is required. Install from [python.org](https://python.org) if needed.
</Note>
