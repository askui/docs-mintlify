---
title: AskUI-SetFirstAskUIRunner
---


### SYNOPSIS

Configures the runner environment.
It tries to create the "AskUIRunnerUser" user and disables the screensaver for it.
It sets the AskUI Suite as global installation.
It adds and starts the AskUI Controller Auto Recover Service.

### PARAMETERS

- `-ControllerType` | _&lt;String&gt;_ - The type of the controller to recover. (Default: RemoteDeviceController for Python SDK)
Valid values are: 'UIController', 'RemoteDeviceController'
Example: 'UIController' for Typescript ADK (NodeJS), 'RemoteDeviceController' for Python SDK.

### NOTES

This Commandlet is only available on Windows AMD64.

### EXAMPLES

#### EXAMPLE 1

Configures the runner environment.
It tries to create the "AskUIRunnerUser" user and disables the screensaver for it.
It sets the AskUI Suite as global installation.
It adds and starts the AskUI Controller Auto Recover Service with the "RemoteDeviceController" type.

```powershell
AskUI-SetFirstAskUIRunner
```
 
#### EXAMPLE 2

Configures the runner environment.
It tries to create the "AskUIRunnerUser" user and disables the screensaver for it.
It sets the AskUI Suite as global installation.
It adds and starts the AskUI Controller Auto Recover Service with the "UIController" type.

```powershell
AskUI-SetFirstAskUIRunner -ControllerType "UIController"
```
