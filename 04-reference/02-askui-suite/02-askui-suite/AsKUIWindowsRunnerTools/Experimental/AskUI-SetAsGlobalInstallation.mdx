---
title: AskUI-SetAsGlobalInstallation
---


### SYNOPSIS

Sets the AskUI Suite as global installation.
It adds the AskUI Suite tools to the system's PATH environment variable.
It also sets the ASKUI_INSTALLATION_DIRECTORY and ASKUI_COMPONENT_REGISTRY_FILE environment variables.
It also adds the Powershell 7 binary to the system's PATH environment variable if it is not already present.

### NOTES

This Commandlet is only available on Windows AMD64.

### EXAMPLES

#### EXAMPLE 1

Sets the AskUI Suite as global installation.

```powershell
AskUI-SetAsGlobalInstallation
```
